基础语法

缩进

缩进就是每一行开头都要有一定的格式，python需要严格的缩进去控制代码

多行语句

如果语句太长会导致代码十分的不整齐并且混乱，我们可以通过 \ 来将同一行语句，变成多行

注释

引号

单引号表示一个词，双引号表示一个句子，六个双引号表示多个句子组成的段落

同行多语句

用分号隔开即可

字面量

定义：通俗来讲就是写在程序里面的值

值类型

类型 描述 说明/举例
数字（Number） 包括：
整数（int）浮点数（float/double）
复数（complex） 布尔（bool）		 int: 10,-8,19
float/double: 1.109,-838.1
complex: 4+3j 以j结尾
bool: 只有true和false，两个值，方便表示真假判断
字符串（String） 相当于一段文本 string:由任意字符组成
列表（List） 有序的可变序列 最常用的类型，可有序记录一堆数据
元组（Tuple） 有序的不可变序列 可有序记录一堆不可变的数据集合
集合（Set） 无序不重复组合 可无序记录一堆不重复的数据集合
字典（Dictionary） 无序的Key-Value集合 可无序记录一堆Key-Value的数据集合

输入语句

通用格式为：变量名 = input(“提示字符串”)

注意：python输入默认解析为字符串，需要进行类型转换，后面会讲

输出语句

输出括号内的内容

print格式化输出：

与C语言的占位符规则相似

占位符 含义
%d 输出整数
%s 输出字符串
%c 输出单个字符
%f 输出小数

当然了这种输出方式可以更方便控制输出格式，至于更详细的用法可以上网查一下，这里不展开了

我们还可以在字符串前声明一个 f 这样也可以格式化输出

这种方式是最优雅的，但是不如占位符好控制，两者可以根据需求进行选择

变量

通俗来讲，就是程序运行中用来存储某种数据的值

比如我想存一个人的身高，可以定义 height = 185 或者 h = 185 ，命名只能有字母、数字、下划线

至于是具体命名更好，还是简单命名更好，我个人建议，如果你是在参加个人竞赛，那么保证命名自己理解就好。如果你是需要与别人合作，比如团队竞赛，或者开发，那么越具体越好

数据类型

可以使用type() 语句来查看数据类型

或者也可以将变量填入

当然了也可以把type() 存入变量输出，这里不演示了

数据类型转换

目标类型(变量)

直接按照这个格式就可以进行类型转换了

运算符

设 a = 10,b = 20

运算符 描述 实例
+ 将数据相加 a + b = 30
- 将数据相减 a - b = -10
* 将数据相乘 a * b = 200
/ 将数据相除，由于python有高精度所以会自动保留小数 b / a = 2
% 取模 - 返回除法的余数 b % a = 0
** 返回x的y次幂 a ** b = 10的20次幂
// 取整除 - 返回商的整数部分（向下取整） 10 // 3 = 3

条件语句

条件语句是通过一条或多条语句的执行结果（True或者False）来决定执行的代码块

Python程序语言指定任何非0和非空（null）值为true，0 或者 null为false

Python 编程中 if 语句用于控制程序的执行，基本形式为：

条件为真时执行if语句下的代码，否则执行else下的代码

当然了，这种只能判断一种条件显然是不够用，所以我们需要完整版的条件语句

注意：判断条件时由上到下判断，只要满足条件直接执行，剩余的条件不会进行判断

判断条件可以用>（大于）、<(小于)、==（等于）、>=（大于等于）、<=（小于等于）来表示其关系

如果判断需要多个条件需同时判断时，可以使用 or （或），表示两个条件有一个成立时判断条件成功；

使用 and （与）时，表示只有两个条件同时成立的情况下，判断条件才成功

循环语句

循环语句主要用于，我需要满足一个条件，当这个条件不满足时，我需要一直重复一段代码直到满足条件

while循环

与条件语句基本一直，只不过在不满足条件时会一直循环执行代码

循环语句还可以嵌套条件语句使用

当然了循环是可以无限的，只要让条件一直为真，用户可以手动用 CTRL+C 来中止循环，这个就不演示了，可以自己实验一下

for循环

循环语句逻辑相同，只是语法不同，用于不同的使用场景逻辑，for循环的逻辑更有针对性，他更适用于对数据进行遍历处理

for循环会逐个将数据集中的数据赋值给临时变量

注意：for循环无法自定义条件，也无法无限循环

上面所称的数据集实际上是序列，例如字符串、元组、列表等等

range语句

range可以构建一个简单序列

嵌套循环

while嵌套

我们可以通过两层循环来写一个九九乘法表，外层控制行数，第二层控制列数

for嵌套

依旧是外层控制行数，分成上下三角打印，内层分别控制空格和星号

循环控制语句

break

若在循环中执行该语句，那么不论如何都会直接跳出整个循环

continue

若在循环中执行该语句，则直接跳过该语句下面所有代码，执行下一次循环
